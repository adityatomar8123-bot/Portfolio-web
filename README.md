# Aditya Tomar — Portfolio ◈

Welcome to the repository for my personal portfolio website. This project is built to deliver a cinematic, scroll-driven experience that showcases my work, skills, and journey as a full-stack developer.

🔗 **Live Site:** [adityastack-dev.vercel.app](https://adityastack-dev.vercel.app/)

## ✨ Features

- **Cinematic Scroll Experience:** Implements a custom Canvas-based, frame-by-frame video sequence that progresses as you scroll, creating a highly immersive background.
- **Premium Glassmorphism UI:** Sleek, frosted-glass components with dynamic borders and interactive hover states.
- **Micro-Interactions & Animations:** High-quality transitions powered by custom CSS and Intersection Observers (magnetic buttons, glowing borders, custom cursors).
- **Interactive "Mindset" Card:** A responsive 3D flip card in the hero section revealing a personalized "developer terminal" on the back.
- **Fully Responsive:** Optimized for seamless performance on both desktop and mobile devices.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Library:** [React](https://reactjs.org/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** CSS Modules / Vanilla CSS / Tailwind CSS
- **Animations:** Custom Canvas Frame Blending, CSS Keyframes
- **Deployment:** [Vercel](https://vercel.com)

## 🚀 Running Locally

Want to explore the codebase or run the site locally? Follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/adityatomar8123-bot/Portfolio-web.git
   cd Portfolio-web
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or yarn install, pnpm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the application running.

## 📂 Project Structure

- `src/components/` - Reusable UI components (Hero, About, Projects, Experience, Contact, FlipCard, etc.)
- `public/` - Static assets including images, SVG icons, and the high-res video frame sequence for the background.
- `src/app/` - Next.js App Router pages and global layouts.
